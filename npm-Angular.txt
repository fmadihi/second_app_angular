برای انگولار باید:
در گوگل بزن:
angular cli
و در سایت خود انگولار دستور زیر را میبینی و میزنی:
npm install -g @angular/cli

بعد از اینکه دستور بالا را زدید و نصب شد یه پوشه برای پروژه انگولار خود بسازید و وارد محیط وی اس کد شوید 
در ترمینال دستور زیر را بزنید:
ng new 
 بعد به سوال هایی که میپرسد پاسخ دهید.

بعد از ساخته شدن دستور
ng serve
را بزنید

برای آپدیت پکیج های نود و وابستگی های اون از دستور زیر استفاده میکنیم:
npm i npm-check-updates

بعد از نصب دستور بالا با استفاده از دستور زیر 
ncu
میاد و براتون پروژه رو آنالیز میکنه و میگه کدوم ها نیزا به به روز رسانی دارند.

حتما دستور بالا را در مسیر پروژه بزنید.

و بعد که میخواهید به روزرسانی کنید دستور زیر را بزنید
ncu -u

دستور بالا باعث میشه که پکیج شما به روزرسانی بشه و حالا برای نصب باید 
npm install
رو بزنید.
ncu عدد ورژن ها را در فایل package.json تغییر میده

---------------
ساخت کامپوننت با cli
ng g c NAme

------------
نصب بوت استرپ

npm install bootstrap

آن را در angular.json فراخوانی میکنیم
در بخش style
"styles": [
              "node_modules/bootstrap/dist/css/bootstrap.min.css",
              "src/styles.css"
            ],

html-css
